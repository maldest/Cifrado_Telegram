# Encriptador Bot de Telegram 🔐

Este bot permite cifrar y descifrar mensajes usando AES (Advanced Encryption Standard) en modo CBC. Todo ocurre dentro de Telegram de forma segura.

## 🤖 Usuario del bot

Puedes hablar con el bot aquí: [@encriptador_bot](https://t.me/encriptador_bot)

---

## 🚀 Instrucciones para ejecutar el bot

1. **Clona o copia este repositorio.**
2. **Instala los requerimientos del entorno** con pip:

```bash
pip install -r requirements.txt
```

---
Ejecuta el archivo
python cifrado_telegram_final.py
